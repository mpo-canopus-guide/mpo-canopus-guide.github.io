# MPO Canopus v12 Workflow Guide

**Complete Guide to Asteroid Lightcurve Analysis**  
*From Raw FITS to Publication-Ready Results*

## About This Guide

This comprehensive guide provides a step-by-step workflow for asteroid lightcurve photometry using MPO Canopus Version 12. It covers the complete pipeline from image loading and quality assessment through photometric measurement, period analysis, and ALCDEF export.

There's currently no good online guide to MPO Canopus v12 available, so this community-created resource fills that gap.

This guide assumes you have:
- Basic familiarity with astronomical imaging and FITS file format
- Calibrated images (bias/dark/flat corrected)
- A basic understanding of differential photometry principles
